Hidden Columns
  • Completely removed from API responses
  • Column won't appear in schema discovery
  • Use for highly sensitive data
  • Example: API keys, passwords, internal IDs
Masked Columns
  • Column appears in schema but returns NULL
  • Maintains API structure
  • Use for less sensitive data
  • Example: Contact info, addresses
