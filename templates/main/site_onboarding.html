      📋 Site Onboarding Help & Guidelines
      🏢 Asset List Table

      Manages basic asset information including capacity, location, and connection details.

      • Asset Code: Unique identifier for the asset
      • Capacity: Site capacity in kWh
      • Coordinates: Latitude and longitude for mapping
      • Timezone: Format: "+05:30" or "-08:00"
      📱 Device List Table

      Contains detailed device information for equipment monitoring.

      • Device ID: Unique device identifier
      • Device Type: Category of equipment
      • Parent Code: Reference to parent asset or device
      • Location: Precise coordinates for device placement
      🔗 Device Mapping Table

      Defines the relationship between devices and data metrics.

      • OEM Tag: Original equipment manufacturer tag
      • Metric: Standardized measurement name
      • Data Type: Type of data being collected
      • Units: Measurement units (kW, kWh, V, A, etc.)
      🔧 Best Practices
      • Always download existing data before uploading new files
      • Validate CSV headers match the expected format
      • Test with small batches first
      • Keep backup copies of original data
      • Contact IT support for large-scale imports
