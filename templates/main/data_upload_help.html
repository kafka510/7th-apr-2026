📋 Data Upload Help & Templates

📤 Data Upload
📝 CSV Format Requirements
  • Files must be in CSV format (.csv extension)
  • First row should contain column headers
  • Column names are case-insensitive and spaces are automatically converted to underscores
  • Date fields should be in YYYY-MM-DD format
  • Numeric fields should contain valid numbers
  • File encoding: UTF-8, Latin-1, or Windows-1252 (the system will automatically detect the encoding)
⚠️ Encoding Issues

If you encounter encoding errors like "utf-8 codec can't decode byte", try these solutions:

  • Open the CSV file in a text editor (like Notepad++) and save it as UTF-8
  • If using Excel, save the file as "CSV UTF-8 (Comma delimited)"
  • Ensure the file doesn't contain special characters that might cause encoding issues

Required columns: country, portfolio, month

📅 Month Format:

The month column should be in the format: "25-Apr", "25-May", etc.

  • First part: 2-digit year (e.g., "25" for 2025)
  • Second part: 3-letter month abbreviation (Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec)

📤 Upload Modes
📈 Append Mode

Add new data to existing records. Duplicates are skipped if "Skip duplicates" is enabled.

🔄 Replace Mode

Delete existing data in the specified date range and replace with new data.

⚙️ Advanced Options
  • Skip duplicates: Prevents duplicate records from being imported
  • Validate data: Checks data format and required columns
  • Batch size: Number of records processed at once (100-10000)
